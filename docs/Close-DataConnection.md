# Close-DataConnection

## SYNOPSIS

Closes a database connection.

## SYNTAX

```
Close-DataConnection [-Connection] <DbConnection> [<CommonParameters>]
```

## DESCRIPTION

The Close-DataConnection cmdlet closes a database connection.
